## 二、MVC 五层架构体现

以已完成的「商品列表」功能为例，展示五层调用链：

```
┌─────────────────────────────────────────────────────────┐
│ Layer 1: View（表现层）                                  │
│   frontend/src/views/goods/List.vue                     │
│   → 用户点击"商品广场"，调用 getGoodsList()               │
│   → 收到数据后渲染商品卡片网格                             │
├─────────────────────────────────────────────────────────┤
│ Layer 2: Controller（控制器层）                          │
│   backend/.../Controllers/GoodsController.cs             │
│   → [HttpGet] 路由 /api/goods                            │
│   → 接收 query params (page, size)                       │
│   → 调 IGoodsService.GetListAsync()                      │
│   → 返回序列化为 JSON 的 ApiResponse<GoodsListResult>    │
├─────────────────────────────────────────────────────────┤
│ Layer 3: Service（业务逻辑层）                            │
│   backend/.../Services/GoodsService.cs                   │
│   → 调用 IGoodsRepository 取数据                          │
│   → 数据库中文状态值 → 前端英文枚举映射                    │
│   → Entity → DTO 转换                                    │
│   → 不碰 SQL，不碰 HTTP                                  │
├─────────────────────────────────────────────────────────┤
│ Layer 4: Repository（数据访问层）                         │
│   backend/.../Repositories/GoodsRepository.cs            │
│   → SELECT * FROM v_goods_list                           │
│   → OFFSET ... ROWS FETCH NEXT ... ROWS ONLY（分页）     │
│   → Oracle 参数化查询                                    │
│   → 只跟数据库打交道，不知道 HTTP / 业务规则               │
├─────────────────────────────────────────────────────────┤
│ Layer 5: Model（数据库层）← DBA 负责                      │
│   database/ddl/003_views.sql → v_goods_list 视图          │
│   → 自动 JOIN app_user（卖家名）+ category（分类名）       │
│   → 自动取 goods_image 第一张作为 cover_image              │
│   → 封装复杂联表，Repository 像查单表一样简单               │
│   → 只过滤 approved + sold 状态，屏蔽待审/下架商品          │
└─────────────────────────────────────────────────────────┘

关键原则：
  Controller 不知道 SQL
  Service 不知道 HTTP
  Repository 不知道业务规则
  View 只跟 Controller 通信
  数据库封装层向上屏蔽表结构细节
```

**Repository 调用示例（C#）：**

```csharp
public async Task<int> RegisterAsync(User entity)
{
    const string sql = @"
        BEGIN
            sp_register_user(
                p_username => :username,
                p_password => :password,
                p_nickname => :nickname,
                p_phone    => :phone,
                p_email    => :email,
                p_user_id  => :user_id
            );
        END;";

    await using var connection = await _connectionFactory.CreateOpenConnectionAsync();
    await using var command = new OracleCommand(sql, connection) { BindByName = true };
    command.Parameters.Add(new OracleParameter("username", OracleDbType.Varchar2, 50) { Value = entity.Username });
    command.Parameters.Add(new OracleParameter("password", OracleDbType.Varchar2, 64) { Value = entity.Password });
    command.Parameters.Add(new OracleParameter("nickname", OracleDbType.Varchar2, 50) { Value = entity.Nickname });
    command.Parameters.Add(new OracleParameter("phone", OracleDbType.Varchar2, 20) { Value = entity.Phone ?? (object)DBNull.Value });
    command.Parameters.Add(new OracleParameter("email", OracleDbType.Varchar2, 100) { Value = entity.Email ?? (object)DBNull.Value });

    var userIdParam = new OracleParameter("user_id", OracleDbType.Int32)
    {
        Direction = ParameterDirection.Output
    };
    command.Parameters.Add(userIdParam);

    await command.ExecuteNonQueryAsync();
    return Convert.ToInt32(userIdParam.Value?.ToString());
}
```

**异常处理：** 用户名重复 → `ORA-20001: 用户名已存在`

```csharp
public async Task<(List<Goods> list, int total)> QueryAsync(
    string? keyword, int? categoryId, decimal? minPrice, decimal? maxPrice,
    string? condition, string? sortBy, int page, int size)
{
    var where = new List<string> { "1=1" };
    var parameters = new List<OracleParameter>();

    if (!string.IsNullOrWhiteSpace(keyword))
    {
        where.Add("title LIKE :keyword");
        parameters.Add(new OracleParameter("keyword", OracleDbType.Varchar2, 210)
            { Value = $"%{keyword}%" });
    }
    if (categoryId.HasValue)
    {
        where.Add("category_id = :catId");
        parameters.Add(new OracleParameter("catId", OracleDbType.Int32) { Value = categoryId });
    }
    if (minPrice.HasValue)
    {
        where.Add("price >= :minPrice");
        parameters.Add(new OracleParameter("minPrice", OracleDbType.Decimal) { Value = minPrice });
    }
    if (maxPrice.HasValue)
    {
        where.Add("price <= :maxPrice");
        parameters.Add(new OracleParameter("maxPrice", OracleDbType.Decimal) { Value = maxPrice });
    }
    if (!string.IsNullOrWhiteSpace(condition))
    {
        where.Add("goods_condition = :condition");
        parameters.Add(new OracleParameter("condition", OracleDbType.Varchar2, 20) { Value = condition });
    }

    var orderBy = sortBy switch
    {
        "price_asc" => "price ASC",
        "price_desc" => "price DESC",
        "views" => "view_count DESC",
        _ => "created_at DESC"
    };

    var offset = (page - 1) * size;
    var sql = $@"
        SELECT goods_id, seller_id, seller_name, seller_avatar, category_id,
               category_name, title, price, goods_condition, goods_status,
               view_count, cover_image, created_at
        FROM v_goods_list
        WHERE {string.Join(" AND ", where)}
        ORDER BY {orderBy}
        OFFSET {offset} ROWS FETCH NEXT {size} ROWS ONLY";

    // 同时查 COUNT(*) 求 total...
}
```

**存储过程内部自动完成（一个事务）：**

```

```

## 五、后端开发模板

新增一个业务模块时，按以下文件结构开发（以 Category 为模板）：

```
backend/CampusSecondHand.Api/
├── Models/
│   ├── Entities/Xxx.cs              # 数据库实体（字段名与表列对应）
│   ├── Dtos/XxxDto.cs               # 前端需要的 DTO（字段名与前端类型对应）
│   └── Requests/XxxRequest.cs       # 前端请求体
├── Repositories/
│   ├── Interfaces/IXxxRepository.cs # 数据访问接口
│   └── XxxRepository.cs            # 实现（调视图或存储过程）
├── Services/
│   ├── Interfaces/IXxxService.cs    # 业务逻辑接口
│   └── XxxService.cs               # 实现（Entity → DTO，调 Repository）
├── Controllers/
│   └── XxxController.cs            # [ApiController] + [Route("api/xxx")]
└── Program.cs                       # 注册 DI：AddScoped<IXxxRepository, XxxRepository>()
```

**Repository 层调用数据库封装层的方式：**

```
简单查询    → SELECT ... FROM v_xxx（视图）
权限校验    → SELECT fn_xxx(...) FROM DUAL（函数调用）
计算/工具   → SELECT fn_xxx(...) FROM DUAL（函数调用）
事务操作    → BEGIN sp_xxx(...); END;（存储过程调用）
自动规则    → 触发器自动生效，无需调用
```
